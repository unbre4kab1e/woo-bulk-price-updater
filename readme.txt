A plugin to bulk update product prices in WooCommerce.

== Description ==

Bulk Price Updater is a WooCommerce plugin that allows you to easily update the prices of your products in bulk. You can change the regular and sale prices either by a fixed amount or by a percentage. The plugin also provides an option to update individual product prices with the same flexibility.

== Installation ==

1. Upload the `bulk-price-updater` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the "Bulk Price Updater" menu in the WordPress admin to start using the plugin.

== Frequently Asked Questions ==

= Can I update only sale prices or only regular prices? =

Yes, the plugin provides options to update either regular prices, sale prices, or both.

= Is it possible to increase or decrease prices by a percentage? =

Yes, you can choose to update prices by a fixed amount or by a percentage.

= How do I revert the price changes? =

Currently, the plugin does not support undoing the price changes. It's recommended to back up your data before making bulk changes.
